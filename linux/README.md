Linux Client
========

## Building

Install the following dependencies
```
gtk+-3.0
glib-2.0
libavutil-dev
libswscale-dev
libasound2-dev
libspeex-dev
libjpeg-turbo
```

Run `make`

To install then, use either the `install` or `install-dkms` scripts [depending on the fact your system supports DKMS](./README-DKMS.md).
